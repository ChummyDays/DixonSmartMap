# DixonSmartMap <br /> <br />

"Smart" Map/Navigation application built for Dixon Gallery & Gardens in Memphis, TN <br /> <br />

Uses Google Maps API for loading map data, and Firebase Realtime Database for pulling in landmark data. <br /> <br />
